# 3.14 键盘快捷键

通用

**Ctrl + X** 剪切  
**Ctrl + C** 复制  
**Ctrl + V** 粘贴

**Ctrl + 鼠标左键拖拽** 复制  
**Ctrl + Z** 撤销  
**Ctrl + Y** 重做

**Ctrl + A** 全选

**Ctrl + N** 新项目  
**Ctrl + O** 打开项目  
**Ctrl + S** 保存项目

**Alt + F4** 关闭Construct 2

你也可以通过**Alt**进行菜单栏导航。当你按下Alt，每个选项应该按下的按钮提示会显示出来。

**F4** 从第一层预览项目  
**Ctrl + F4** 从第一层调试  
**F5** 预览当前层（按下shift进行持续预览）  
**Ctrl + F5** 调试当前层  
**F6** 导出项目

**Ctrl + 鼠标滚轮上** 或者 **数字键盘 +** 视角进  
**Ctrl + 鼠标滚轮下** 或者 **数字键盘 -** 视角退  
加shift调整进退距离

**Ctrl + 0** 返回100%视角
